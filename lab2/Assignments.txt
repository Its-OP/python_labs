General:
1. The title on the window must display the action that the window performs.
2. The window must contain brief information about the application author (about 5 lines of text)
3. The window must have a convenient layout of elements.
4. The user must be able to resize (scale up or down) the window. Its child elements must scale along with the window.
5. The window must support two color themes (light and dark), and there must be a button to toggle between them. The color themes can be simple.


Scenario 1: #11
The user enters a line of text. The application should find all the palindrome words in the text and display them in the GUI window. One or more spaces separate the words in the text.

Task 2: #21
The user enters the a, b, and c coefficients of the following equation: 'a*x^2 + b*x + c = 0'. The application must solve the equation and display the solutions. If no real solutions exist, the application must display the 'No real solutions exist' text.

Task 3: #16
The user enters a sentence. The application should invert every even word in the sentence and display the result in the GUI window. The words in the sentence are separated by spaces.
